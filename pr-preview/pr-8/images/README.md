# Images Directory

Place your images here. They will be accessible at `/images/filename.jpg` in your posts.

## Example Usage

In your post, reference images like this:

```markdown
{{< image src="/images/main-image.jpg" alt="Description" caption="Image caption" >}}
```

Or for featured images in front matter:

```toml
featuredImage = '/images/main-image.jpg'
```
